C變為 E[X] + c
D保持不變
✅ 正確答案：B
">期望值具有線性性質。根據期望值的定義 E[cX] = Σ [(c * xi) * P(X=xi)]。由於常數 c 可以提出求和符號外，得到 E[cX] = c * Σ [xi * P(X=xi)] = c * E[X]。也就是說，一個常數乘以一個隨機變數的期望值，等於該常數乘以該隨機變數的期望值。
