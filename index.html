第 1 題 L21 L21101 自然語言處理技術與應用
機器翻譯 (Machine Translation, MT) 按照技術發展階段，主要經歷了哪些範式？
A僅基於規則的機器翻譯
B僅統計機器翻譯
C僅神經機器翻譯
D基於規則的機器翻譯 (RBMT)、統計機器翻譯 (SMT)、神經機器翻譯 (NMT)
✅ 正確答案：D
">機器翻譯的發展歷程主要可以劃分為三個階段： 基於規則的機器翻譯 (Rule-Based Machine Translation, RBMT): 早期的方法，依賴語言學家手動編寫大量的雙語詞典和語法規則來進行翻譯。優點是語法相對準確，缺點是規則覆蓋有限、構建成本高、難以處理歧義和口語。 統計機器翻譯 (Statistical Machine Translation, SMT): 20 世紀 90 年代興起，利用大規模平行語料庫（成對的原文和譯文句子）學習翻譯模型（如詞對齊、短語翻譯機率）和語言模型。相比 RBMT 效果有顯著提升，但模型複雜，且難以處理長距離依賴。 神經機器翻譯 (Neural Machine Translation, NMT): 2014 年左右開始發展，使用深度神經網路（主要是基於 RNN 的 Encoder-Decoder 架構，後來被 Transformer 取代）直接學習從源語言到目標語言的端到端映射。NMT 在翻譯流暢度和準確度上取得了突破性進展，成為當前主流的機器翻譯範式。

第 3 題 L21 L21102 電腦視覺技術與應用
仿射變換 (Affine Transformation) 在圖像處理中可以實現哪些操作？
A僅亮度調整
B平移、旋轉、縮放、錯切 (Shear)
C僅濾波操作
D僅顏色空間轉換
✅ 正確答案：B
">仿射變換是一種線性變換（通過矩陣乘法實現）與平移（向量加法）的組合。在二維圖像處理中，它可以表示為 `x' = Ax + b`，其中 x 是原始坐標，x' 是變換後坐標，A 是一個 2x2 的線性變換矩陣，b 是一個 2x1 的平移向量。仿射變換可以實現多種幾何操作，包括： 平移 (Translation): 移動圖像位置。 旋轉 (Rotation): 繞某點旋轉圖像。 縮放 (Scaling): 放大或縮小圖像（可以沿不同軸進行不同比例縮放）。 錯切 (Shear): 使圖像沿某個方向傾斜（像推倒長方形）。 仿射變換保持了圖像中直線的平行性，但不一定保持角度和長度。它是比剛性變換（僅平移和旋轉）更廣泛的一類變換。
第 4 題 L21 L21101 自然語言處理技術與應用
TF-IDF (Term Frequency-Inverse Document Frequency) 是一種常用的文本表示方法，其中 IDF (Inverse Document Frequency) 的主要作用是什麼？
A計算一個詞語在單一文件中出現的頻率
B降低在許多文件中都普遍出現的詞語的權重，提高在少數文件中出現的詞語的權重
C將詞語轉換成固定長度的向量
D計算文件之間的相似度
✅ 正確答案：B
">TF-IDF 是一種統計方法，用以評估一個詞語對於一個文件集或一個語料庫中的一份文件的重要程度。TF (Term Frequency) 指的是詞語在文件中的出現頻率。IDF (Inverse Document Frequency) 則衡量詞語的普遍性。如果一個詞語在很多文件中都出現，它的 IDF 值會較低；反之，如果只在少數文件中出現，IDF 值會較高。TF 和 IDF 相乘得到 TF-IDF 值，這個值可以突顯那些在特定文件中頻繁出現但在整個語料庫中相對罕見的詞語，這些詞語通常更能代表該文件的內容。因此，IDF 的主要作用是降低普遍詞（如停用詞）的影響力，提升具有區別性詞語的重要性。
第 5 題 L21 L21104 多模態人工智慧應用
多模態AI模型可能繼承訓練數據中的社會偏見（Social Bias），例如在圖像描述生成中對不同性別或膚色的人產生刻板印象的描述。應對這個挑戰的策略不包含？
A收集更多樣化、更平衡的訓練數據。
B設計去偏見（Debiasing）的演算法或正則化項。
C對模型的輸出進行後處理（Post-processing）以修正偏見。
D完全避免使用從真實世界收集的數據。
✅ 正確答案：D
">演算法偏見是AI倫理中的一個重要議題，多模態AI也不例外。由於訓練數據來源於充滿偏見的現實世界（尤其是網路數據），模型很可能學習並放大這些偏見。應對策略通常包括：(1) 數據層面：努力收集或構建更多元、更平衡、代表性更廣的數據集，並進行數據審計；(2) 演算法層面：設計能夠減輕偏見影響的模型架構、損失函數或訓練策略；(3) 後處理層面：在模型輸出後進行檢測和修正。完全避免使用真實世界數據是不現實的，因為模型需要從真實數據中學習知識。關鍵在於如何負責任地收集、處理數據和設計模型。
第 6 題 L21 L21201 AI導入評估
模型漂移 (Model Drift) 是 AI 系統維運中的一個潛在風險，它指的是什麼？
A模型程式碼被意外修改
B由於真實世界數據分佈隨時間變化，導致已部署模型的效能逐漸下降的現象
C模型訓練過程中斷
D模型被複製到多個伺服器
✅ 正確答案：B
">AI 模型通常是在某個特定時間點的歷史數據上訓練的。然而，真實世界的環境是動態變化的，例如使用者行為改變、市場趨勢變化、外部事件影響等，這會導致線上實際遇到的數據分佈（輸入數據的特性或目標變數的關係）與訓練時的數據分佈產生差異。這種差異會使得原本效能良好的模型，在部署一段時間後預測能力逐漸下降，這種現象就稱為模型漂移。模型漂移主要有兩種類型：概念漂移 (Concept Drift)（輸入特徵與目標變數之間的關係發生變化）和數據漂移 (Data Drift)（輸入數據本身的統計特性發生變化）。為了應對模型漂移，需要建立持續的監控機制，定期評估模型效能，並在必要時進行模型的重新訓練或更新。

第 8 題 L22 L22103 假設檢定與統計推論
統計推論（Statistical Inference）的主要目的是什麼？
A僅描述樣本數據的特性
B利用樣本數據的資訊，對未知的母體參數或特性做出估計或判斷
C設計收集數據的實驗方法
D創建資料視覺化圖表
✅ 正確答案：B
">統計推論是利用從母體中抽取的樣本數據，來對母體本身的特性（例如母體平均數、母體比例、母體變異數或不同母體之間的差異）進行推斷的過程。主要包含兩大類方法：(1) 參數估計（Parameter Estimation）：使用樣本統計量來估計未知的母體參數，包括點估計（Point Estimation）和區間估計（Interval Estimation，如信賴區間）。(2) 假設檢定（Hypothesis Testing）：根據樣本證據，對關於母體特性的某個假設做出接受或拒絕的判斷。選項 A 是敘述性統計的目的。選項 C 和 D 是數據分析過程中的其他環節。

第 14 題 L21 L21102 電腦視覺技術與應用
卷積神經網路 (Convolutional Neural Network, CNN) 在電腦視覺領域取得巨大成功的主要原因是？
A使用了循環連接來處理序列資訊
B通過卷積層的權重共享和池化層的降維，有效地提取圖像的空間階層特徵並減少參數數量
C完全依賴於人工設計的特徵
D使用了強化學習來優化圖像處理流程
✅ 正確答案：B
">CNN 的設計借鑒了生物視覺皮層的處理機制，其核心優勢在於： 局部感受野 (Local Receptive Fields): 卷積核（濾波器）只關注輸入圖像的一小塊區域，提取局部特徵。 權重共享 (Weight Sharing): 同一個卷積核在圖像的不同位置滑動，使用相同的權重參數。這大大減少了模型的參數數量（相比全連接網路），降低了過擬合風險，並使得模型對物體的平移具有一定的不變性。 池化層 (Pooling Layers): 通常在卷積層之後使用，通過降採樣（如最大池化 Max Pooling 或平均池化 Average Pooling）來降低特徵圖的空間維度，進一步減少計算量和參數，同時增強模型對微小位移和形變的穩健性。 階層式特徵學習 (Hierarchical Feature Learning): 淺層卷積層通常學習邊緣、角點等低階特徵，深層卷積層則基於低階特徵組合出更複雜、更抽象的高階特徵（如物體部件、完整物體）。 這些特性使得 CNN 非常適合處理具有空間結構的圖像數據，能夠自動學習到有效的視覺特徵表示。選項 A 描述的是 RNN。選項 C 是傳統電腦視覺方法的特點。選項 D 描述的是強化學習。
