第 1 題 L22 L22102 機率分佈與資料分佈模型
機率密度函數（Probability Density Function, PDF）f(x) 對於連續隨機變數 X 的主要作用是？
A直接給出 X 取某個特定值 x 的機率 P(X=x)
B描述 X 在不同數值點附近的相對可能性，其在某個區間 [a, b] 上的積分等於 X 落在該區間的機率 P(a ≤ X ≤ b)
C計算 X 的期望值 E[X]
D計算 X 的變異數 Var(X)
✅ 正確答案：B
">對於連續隨機變數，其取任何一個特定單點值的機率理論上為 0（A 錯誤）。我們關心的是變數落在某個區間內的機率。機率密度函數 PDF f(x) 本身並非機率值，但它描述了變數在數值 x 附近單位寬度內的機率密度（相對可能性）。要計算變數 X 落在區間 [a, b] 內的機率 P(a ≤ X ≤ b)，需要計算 PDF 曲線在該區間下方的面積，即對 f(x) 從 a 到 b 進行積分（∫[a,b] f(x) dx）（B 正確）。雖然 PDF 可以用來計算期望值（∫ x*f(x) dx）和變異數（∫ (x-μ)^2*f(x) dx），但其主要作用是定義和計算區間機率。

第 4 題 L21 L21102 電腦視覺技術與應用

A單張圖像的陰影
B來自兩個或多個不同視點的圖像之間的視差 (Disparity)
C物體的顏色
D圖像的紋理
✅ 正確答案：B
">立體視覺模仿人類雙眼視覺系統。它使用兩個（或多個）在不同位置拍攝的相機來捕捉同一場景的圖像。由於相機位置不同，同一個三維空間點在兩張圖像上的投影位置會存在差異，這個差異稱為視差 (Disparity)。視差的大小與該點的深度（距離相機的遠近）成反比：距離越近的點，視差越大；距離越遠的點，視差越小。通過找出兩張圖像中對應點（稱為同名點匹配 Stereo Matching）並計算它們之間的視差，就可以利用三角測量原理推斷出該點的三維深度資訊，進而重建場景的三維結構。

B首次將 Transformer 應用於圖像分類
C引入了殘差連接 (Residual Connection) 或跳躍連接 (Skip Connection)，允許資訊直接跨層傳遞，有效解決了深度網路訓練中的梯度消失和退化問題
D使用了分組卷積 (Grouped Convolution) 來減少計算量
